Welcome to the **News Application** documentation!
==================================================

The News Application is a modern Django-based platform for publishing, managing, and sharing news articles, newsletters, and more. It features user authentication, editorial workflows, REST APIs, and beautiful documentation.

.. image:: https://img.icons8.com/ios-filled/100/000000/news.png
   :alt: News Icon
   :align: center

**Features:**

- User registration, login, and permissions
- Article creation, editing, approval, and deletion
- Newsletter management
- REST API with Django REST Framework
- Sphinx documentation
- Docker support
